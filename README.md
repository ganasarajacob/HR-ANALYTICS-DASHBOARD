# HR Analytics Dashboard Project

## Project Overview
A comprehensive HR Analytics project designed to track employee performance, attrition, and demographics. This project utilizes multiple business intelligence tools to deliver actionable insights.

## Datasets Used
* **HR1 Dataset:** Contains employee demographic data.
* **HR2 Dataset:** Contains employee performance and payroll metrics.



## Tools & Technologies Used
* **Excel:** Initial data cleaning and quick pivot table dashboards.
* **Power BI:** Interactive reporting, data modeling, and DAX calculations.
* **Tableau:** Advanced data visualisations and storyboards.
* **PowerPoint:** Executive summary presentation for stakeholders.
